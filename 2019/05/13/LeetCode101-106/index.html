LeetCode 101 对称二叉树

Description

给定一个二叉树，检查它是否是镜像对称的。

例如，二叉树 [1,2,2,3,4,4,3] 是对称的。

但是下面这个 [1,2,2,null,3,null,3] 则不是镜像对称的:

说明:

如果你可以运用递归和迭代两种方法解决这个问题，会很加分。

Analysis

用递归很简单，以根节点为分割，判断左子树和右子树是否对称，即节点1左子节点和节点2右子节点值是否相等，节点1右子节点和节点2左子节点是否相等，若相等依次递归。

迭代法则是遍历出每一层的节点，判断该层节点值是否是对称的。

LeetCode 104 二叉树的最大深度

Description

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：
给定二叉树 [3,9,20,null,null,15,7]，

返回它的最大深度 3 。

Analysis

容易想到递归的DFS解法：递归查找每个节点的左子节点和右子节点，如果该节点不为Null，则深度加一，返回左子树和右子树深度的最大值。

同时也可以使用BFS迭代求解，遍历每层节点，当该层不为空时深度+1.

Solution

Description

根据一棵树的前序遍历与中序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

Analysis

根据前序遍历和中序遍历的定义，前序遍历的第一个节点是根节点，则中序遍历中该节点左侧就是它的左子树，右侧就是它的右子树。这样就可以计算出一个顶点对应的左子树和右子树的长度。前序遍历中根节点的下一个节点就是左子节点，右子节点就是根节点加上左子树长度对应的节点。比如例子中的顶点的左子树长度是1，右子树长度是3.回到前序遍历中，对应的左子树的顶点就是preorder[1]=9, 右子树的顶点就是preorder[2]=20.

接下来递归左子树和右子树即可。例子中9为顶点的情况下左子树右子树长度均为0，因此它的左右子节点均为null.20为顶点的情况下在中序遍历中找到它的左子树长度为1，右子树长度为1，依次遍历即可。

该题最重要的一点就是树中没有重复的元素。

LeetCode 106 从中序与后序遍历序列构造二叉树

Description

根据一棵树的中序遍历与后序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

例如，给出

返回如下的二叉树：

Analysis

根据中序遍历的定义（左根右）、后序遍历的定义（左右根），后序遍历中的最后一个节点必为根节点，根据该节点在中序遍历中的位置划分为左子树和右子树。同理后序遍历中根节点前面的一个节点必为右子节点，从右子树长度（中序遍历中计算出）可以得到左节点的下标。

在例子中，根节点是postorder[4]=3, 右子节点是postorder[4-1]=20,根据中序遍历得左子树长度=1，右子树长度=3，则左子节点是postorder[4-3-1]=9.

接下来得到9不再有子节点，右节点20的右节点为postorder[3-1]=7, 左子树长度为1，右子树长度为1，递归即可。

该题最重要的一点也是树中没有重复的元素。
